# Закрывающий тег

Проект представляет собой адаптивную страницу-воспоминание о пути обучения фронтенд-разработке. На странице собраны этапы прохождения курса, оформленные в виде карточек с изображениями, личными заметками и анимациями.

## Описание проекта

На странице есть:

- адаптивная шапка проекта;
- восемь тематических карточек с личными заметками;
- декоративные лейблы поверх изображений;
- различные CSS-фильтры для изображений;
- анимированные кнопки и иконка лайка;
- модальное окно на основе элемента dialog;
- состояния hover, focus и focus-visible для интерактивных элементов.

Проект свёрстан по макету Figma с использованием современных возможностей HTML и CSS.

## Технологии

- HTML5
- CSS3
- Flexbox
- CSS Grid
- CSS-переменные
- CSS-анимации
- CSS-фильтры
- SVG-иконки
- Dialog API
- Media Queries
- Адаптивная вёрстка
- Подключение локальных шрифтов через @font-face

## Адаптивность

Проект оптимизирован под основные размеры экранов:

- 375px — мобильная версия;
- 1440px — десктопная версия.

Для разных разрешений изменяются:

- размеры и отступы блоков;
- расположение элементов интерфейса;
- размеры кнопок и иконок;
- отображение модального окна.

## Анимации и эффекты

В проекте реализованы:

- анимация кнопок при наведении;
- плавные состояния фокуса;
- анимация иконки сердца;
- эффект появления искр после лайка;
- плавное изменение цветов элементов;
- CSS-фильтры для изображений карточек.
